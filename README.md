# 🍯 Decline in U.S. Honey Production: SQL & Data Analysis

## 📊 Project Overview

This project analyzes trends in U.S. honey production over time using SQL and Python. The goal is to identify long-term declines, state-level differences, and patterns that may explain changes in honey production.

## 🎯 Objectives

* Analyze honey production trends from 1995–2021
* Identify decreases in production over time
* Explore relationships between colonies, yield, and production
* Visualize insights using interactive charts

## 🛠️ Technologies Used

* SQL (SQLite)
* Python (Pandas, Matplotlib)
* Kaggle Notebooks
* Flourish (Data Visualization)

## 📥 Dataset

* Source: Kaggle
* Dataset: U.S. Honey Production (1995–2021)

## 🔍 Key Analysis

* Total honey production per year
* Trends in colony numbers and yield per colony
* Identification of long-term production decline
* Data cleaning and transformation using Pandas
